# DOCLAB PRO v2.0 — Document Processing Suite

## What's new in v2
- Bug fixes: all conversions now properly handle file reads & downloads
- New: PDF → Text (.txt) extraction
- New: Image Compress (JPG/PNG size reduction with quality slider)
- New: Real PDF Merge, Split, Compress, Rotate, Watermark (fully functional)
- New: Dark / Light theme toggle
- New: Toast notifications for success/error
- New: Drag-to-reorder files (for Merge tool, multi-image PDF)
- Improved: Mobile-responsive layout

## Quick Start
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Tools
**Convert to PDF:** Image, Word, PPT, Excel, HTML
**Convert from PDF:** JPG, Text, Word, PPT, Excel, PDF/A
**PDF Tools:** Merge, Split, Compress, Image Compress, Protect, Unlock, Rotate, Watermark, OCR
